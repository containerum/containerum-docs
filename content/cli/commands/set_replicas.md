
### set replicas

**Aliases**   :

re, rep, repl, replica

**Usage**     :

Set deployment replicas.

**Example**   :

chkit set replicas [-n namespace_label] [-d depl_label] [N_replicas]

**Flags**     :

  + deployment d : deployment name
  + replicas r : replicas, 1..15
