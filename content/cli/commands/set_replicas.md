### set replicas

**Description**:

Set deployment replicas.

**Example**:

chkit set replicas [-n namespace_label] [-d depl_label] [N_replicas]

**Flags**:

| Short | Name | Usage | Default value |
| ----- | ---- | ----- | ------------- |
| -d | deployment | deployment name |  |
| -r | replicas | replicas, 1..15 | 1 |
