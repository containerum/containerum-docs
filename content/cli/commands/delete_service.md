### delete service

**Description**:

Delete service in namespace.

**Example**:

chkit delete service service_label [-n namespace]

**Flags**:

| Short | Name | Usage | Default value |
| ----- | ---- | ----- | ------------- |
| -f | force | force delete without confirmation | false |
