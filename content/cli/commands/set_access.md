
### set access

**Aliases**   :

namespace-access, ns-access

**Usage**     :

Set namespace access rights.
Available access levels are:
  none
  owner
  read
  read-delete
  write

**Example**   :

chkit set access $USERNAME $ACCESS_LEVEL [--namespace $ID]

**Flags**     :

  + force f : suppress confirmation
