### get pod

**Description**:

Show pod info.

**Example**:

chkit get pod pod_label [-o yaml/json] [-f output_file]

**Flags**:

| Short | Name | Usage | Default value |
| ----- | ---- | ----- | ------------- |
| -f | file | output file | - |
| -o | output | output format (json/yaml) |  |
