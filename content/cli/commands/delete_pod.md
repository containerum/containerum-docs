### delete pod

**Description**:

Delete pods.

**Example**:

chkit delete pod pod_name [-n namespace]

**Flags**:

| Short | Name | Usage | Default value |
| ----- | ---- | ----- | ------------- |
| -f | force | delete pod without confirmation | false |
