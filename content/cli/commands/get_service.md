### get service

**Description**:

Show service info.

**Example**:

chkit get service service_label [-o yaml/json] [-f output_file]

**Flags**:

| Short | Name | Usage | Default value |
| ----- | ---- | ----- | ------------- |
| -f | file | output file | - |
| -o | output | output format [yaml/json] |  |
