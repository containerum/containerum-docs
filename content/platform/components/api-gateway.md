# Gateway
Gateway is an API service that provides routing for Containerum components.

### Prerequisites

- Kubernetes

### Installation

Using Helm:

```
  helm repo add containerum https://charts.containerum.io
  helm repo update
  helm install containerum/api-gateway
  ```

  Now you can proceed to [installing the user-manager component](/platform/components/user-manager).
